# kata

> A personal problem archive — one problem at a time.

Built with [Quarto](https://quarto.org). No fluff.

## What's inside

Numbered problems accumulated daily (001, 002, …).  
Categories: algorithms, single-cell biology, research design.

Each entry covers:

- **Problem** — the question, clearly stated
- **Solution** — worked answer with reasoning
- **Pattern** — the underlying idea worth remembering

## Structure

```
kata/
├── quiz/         # numbered problems (001.qmd, 002.qmd, …)
├── solution/     # numbered answers (001.qmd, 002.qmd, …)
├── _quarto.yml   # site config
└── index.qmd     # home
```

## Usage

```bash
quarto preview
quarto render
```
